# Desenvolvimento do Projeto

- TP1
Foi criado um arquivo de credenciais para acessar as chaves da API, que permanecerá oculto no GitHub.
O arquivo topmusics.py foi desenvolvido para executar as tarefas do TP1 como o inicial modelo do projeto.

- TP2
O arquivo topmusics.py foi aprimorado e novos arquivos foram adicionados: analise.py, dados_scraping.py e main.py.
analise.py foi criado para executar as tarefas do TP2.
dados_scraping.py foi implementado para extrair conteúdo da web utilizando a biblioteca Beautiful Soup e armazenar os dados em formato CSV na pasta data.
main.py foi desenvolvido para gerenciar a paginação, facilitando a navegação e o uso do projeto.
